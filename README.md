# GA Lottery Scout 🎟️

Real-time Georgia lottery scratch-off tracker. Find the hottest tickets with the most top prizes claimed.

**Monetization:** Free tier + Premium subscription ($4.99/month)

## Features

### Free Tier
- Daily updated scratch-off rankings
- Historical top prize data
- Basic ticket search

### Premium ($4.99/month)
- Real-time alerts (new top prizes claimed)
- Email/SMS notifications
- Advanced analytics
- Predictive scoring

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000
